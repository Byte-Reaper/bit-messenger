# bit-messenger

envrypted chat system where you can host your own server

# SERVER-side setup

1. download the server.py file
2. run the file and the address will show up (if you're using a firewall, remember to allow python through the firewall)
3. share the address with your friends

# CLIENT-side setup

1. download the client.py file
2. run the file and it will ask for the ip and port, if your friend sent you something like this '192.148.94.3:666' then the '192.148.94.3' is the ip and '666' is the port
3. start chatting with your friends
