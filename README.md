# bit-messenger

envrypted chat system where you can host your own server

# SERVER setup

1. download all the files (except for 'client.py' and 'start client.bat' if you only need a server)
2. make shure python is installed and on PATH system variable (if you don't know how to do that, visit https://geek-university.com/python/add-python-to-the-windows-path/)
3. install cryptography with pip
4. make shure python is allowed through firewall
5. run 'start server.bat'

# CLIENT setup

1. download 'client.py' and 'start client.bat'
2. make shure python is installed and on PATH system variable (if you don't know how to do that, visit https://geek-university.com/python/add-python-to-the-windows-path/)
3. install cryptography with pip
4. make shure python is allowed through firewall
5. run 'start client  .bat'
